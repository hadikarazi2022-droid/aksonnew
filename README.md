## Web Application Conversion

This repository has been enhanced with a web application version of the original Akson desktop application. The macOS application has been successfully converted to work as a website, maintaining all core functionality while making it accessible from any device with a web browser.

### Web Application Features

- **Responsive Design**: Works on desktops, tablets, and mobile devices
- **Cross-Platform Access**: Access your study materials from anywhere
- **Preserved Functionality**: All core features from the desktop application
- **Modern Interface**: Clean, intuitive web-based user experience

### How to Run the Web Application

1. Make sure you're in the root directory of the project
2. Install dependencies: `pip install -r requirements.txt`
3. Navigate to the web application directory: `cd web_app`
4. Start the server: `python app.py`
5. Visit `http://localhost:5000` in your browser
